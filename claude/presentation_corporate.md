**Accès** :
- SSO entreprise (SAML/OAuth)
- RBAC granulaire
- Audit logs complets
- 2FA obligatoire

## Anonymisation Automatique

```python
import re

def anonymiser_pii(texte: str) -> str:
    """Anonymise les données personnelles."""

    # Emails
    texte = re.sub(
        r'\b[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}\b',
        '[EMAIL_MASQUE]',
        texte, flags=re.I
    )

    # Téléphones français
    texte = re.sub(
        r'\b0[1-9](?:[\s.-]?\d{2}){4}\b',
        '[TEL_MASQUE]',
        texte
    )

    # Numéros sécurité sociale
    texte = re.sub(
        r'\b[1-2]\s?\d{2}\s?\d{2}\s?\d{2}\s?\d{3}\s?\d{3}\s?\d{2}\b',
        '[SECU_MASQUE]',
        texte
    )

    return texte
```
:::

| **Scalabilité** | ⚠️ Limitée matériel | ✅ Illimitée |
